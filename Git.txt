Основные команды
ls - список файлов в активной директории
cd Downloads/ - смена активной директории (после нескольких символов кликнуть ТАБ чтобы заполнить автоматически)
cd ~ - возвращает активную директорию по умолчанию (папка пользователя)
cd .. - переход в диреткорию на уровень выше
cd ../.. - переход в директорию на два уровня выше
mkdir TestDir - Создать новую директорию (папку) с указанным именем
touch test.txt - Создает файл с указанным именем в активной директории (папке)
cp Test.txt test_copy.txt - копирование файла и задание ему нового имени
mv test.txt test_original.txt - переименовать файл "из" "в"
mv test_original.txt testdir1/test_original.txt - переместить файл в указанную директорию и задать ему имя
echo "Hello" - вывод сообщения в терминал
echo "Hello" > test_original.txt - записывает текст в файл
cat test_original.txt - показывает содержимое файла
rm test_original.txt - удаление файла
rm -R TestDir1/ - удаление директории и всего содержимого в ней

Git. Репозитории
Типы
Локальные (local)
Удаленные (remote)
git init - создает репозиторию в активной папке
ls -a - показать список файлов в активной директории (в том числе и скрытых)
git status - показывает текущее состояние репозитория
git remote add origin https://github.com/F0XDZR/repo-github.git - добавляет синхронизацию между локальными файлами и сервисом GITHUB с активной папкой в которой был создан репозиторий
git remote -v - проверка того, что предыдущая команда прошла успешно
git add readme.md - добавление файла в отслеживание
git commit - записать коммит
git commit -m 'commit message' - записать коммит с сообщением
